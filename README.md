# Cafe Management System

A desktop-based Cafe Management System developed using JavaFX and MySQL. The application provides functionality for managing cafe operations, including user authentication, inventory management, and menu administration through a graphical user interface.

---

## Features

### User Authentication

* Login system with role-based access
* Separate Admin and User account types
* Secure credential validation through MySQL database

## Project Structure

* Controllers – Handle user interactions and UI events
* Models – Represent application data and business logic
* DAO Classes – Manage database operations
* Views – JavaFX user interface components
* Database – MySQL tables and SQL scripts
